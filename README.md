# Маркетинговая аналитика: Исследование клиентов и оценка эффективности кампаний

## Описание проекта
Практический проект в сфере Customer Analytics, направленный на изучение целевой аудитории, структуры покупок и поиск точек роста для увеличения продаж.

## Какие задачи были решены:
* **Профилирование аудитории:** Анализ социально-демографических признаков клиентов (возраст, пол, регион и др.).
* **Анализ покупательского поведения:** Исследование структуры корзины, частоты покупок и среднего чека.
* **Оценка эффективности маркетинга:** Анализ данных по маркетинговым кампаниям, выявление наиболее успешных каналов привлечения и удержания клиентов.
* **Поиск точек роста:** Формулирование рекомендаций для продуктовой и маркетинговой команды на основе выявленных закономерностей в данных.

## Технологический стек
* **Python** (Pandas, NumPy, Matplotlib / Seaborn для базовой визуализации)
* **Анализ данных:** Когортный/сегментный анализ, расчет маркетинговых метрик.
